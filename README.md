# 🚀 Satellite Debris Mitigation - Game Mode

An interactive, game-like interface for space debris monitoring and mitigation with real-time animations, scoring, and achievements.

## Features

### 🎮 Game System
- **Scoring System**: Earn points for scans, tracking, and successful mitigations
- **Level Progression**: Level up as your score increases
- **Achievement System**: Unlock achievements for various milestones
- **Accuracy Tracking**: Monitor your success rate across all operations

### 🔍 Scan Module
- Real-time orbital debris detection
- Configurable scan range (200-2000 km)
- Adjustable scan speed (Slow/Medium/Fast)
- Visual radar sweep animation
- Risk classification (High/Medium/Low)
- Dynamic debris generation and movement

### 📡 Track Module
- Monitor individual debris objects
- Real-time orbital path visualization
- Telemetry data display (Altitude, Inclination, Period, Velocity)
- Animated orbital trails
- Earth-centered visualization

### 💥 Mitigation Module
- Multiple weapon types (Laser, Kinetic, Capture Net)
- Adjustable power levels (1-100%)
- Target selection system
- Success/failure feedback
- Visual explosion effects

### 📊 Dashboard
- Mission statistics
- Achievement tracking
- Real-time mission log
- Success rate monitoring

## Getting Started

1. Open `index.html` in a modern web browser
2. Use the navigation buttons to switch between modules
3. Start scanning for debris
4. Track high-risk objects
5. Engage and neutralize threats
6. Build your score and unlock achievements!

## Controls

### Scan Module
- **Start Scan**: Begin debris detection
- **Stop Scan**: Halt current scan
- **Scan Range**: Adjust detection distance (km)
- **Scan Speed**: Choose detection speed

### Track Module
- Enter debris ID or use detected objects
- Click "Track Object" to monitor
- View real-time telemetry data

### Mitigation Module
- Select weapon type
- Adjust power level
- Choose target from available list
- Click "Engage Target" to neutralize

## Scoring System

- **Scan Complete**: 100 + (High-risk debris × 50) points
- **Object Tracked**: 75 points
- **Target Destroyed**: 200 + power level points
- **Achievements**: 100-1000 bonus points

## Achievements

- 🔍 **First Scan**: Complete your first debris scan
- 📡 **Scanner**: Complete 10 scans
- 💥 **Debris Destroyer**: Destroy 5 debris objects
- 🏆 **Expert Hunter**: Destroy 20 debris objects
- ⭐ **Score Master**: Reach 10,000 points
- 🎯 **Perfect Accuracy**: Maintain 90%+ success rate
